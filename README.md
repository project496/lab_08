# 🧠 AI Scrabble Word Finder (Streamlit + TensorFlow)

An AI-powered Scrabble word generator that finds valid words from given letters and ranks them using a machine learning model built with TensorFlow.

---

## 🚀 Features

- Enter any set of letters
- Generates all valid Scrabble words
- Uses real dictionary (`words.txt`)
- Calculates Scrabble scores
- AI model ranks best words intelligently
- Streamlit web interface
- Fast and lightweight

---

## 🧠 How It Works

1. User enters letters (e.g., `scrabble`)
2. System checks all possible words from dictionary
3. Filters words that can be formed using given letters
4. Calculates Scrabble score for each word
5. AI model ranks words based on:
   - Letter patterns
   - Word length
   - Vowel ratio
6. Displays top results

---

## 🛠️ Tech Stack

- Python 3.10+
- Streamlit
- TensorFlow (Keras)
- NumPy
